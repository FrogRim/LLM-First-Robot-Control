### 4. 작업 스크립트 작성 ✅

다음 실험을 위한 스크립트 3개 작성:

#### 4.1 `scripts/statistical_analysis.py`
- **기능**: Perplexity 계산, Bootstrap CI 계산, 종합 통계 분석
- **주요 함수**:
  * `calculate_perplexity()`: exp(loss) 계산
  * `bootstrap_confidence_interval()`: n=1000 Bootstrap 샘플링
  * `bootstrap_accuracy_ci()`: 이항 분포 기반 정확도 CI
  * `analyze_model_statistics()`: 종합 통계 보고서 생성

**사용 예시**:
```bash
python scripts/statistical_analysis.py \
    --adapter_dir ./droid-physics-qwen14b-qlora \
    --test_data ./droid_physics_llm_test_alpaca_v2.json \
    --limit 30 \
    --output ./statistical_analysis_results.json
```

#### 4.2 `scripts/baseline_comparison.py`
- **기능**: Fine-tuned model vs Base model 비교
- **평가 지표**:
  * JSON 파싱률
  * 재료 인식 정확도
  * 추론 시간
  * 평균 신뢰도
- **출력**: 개선율(improvement) 계산 및 시각화

**사용 예시**:
```bash
python scripts/baseline_comparison.py \
    --base_model Qwen/Qwen2.5-14B-Instruct \
    --adapter_dir ./droid-physics-qwen14b-qlora \
    --test_data ./droid_physics_llm_test_alpaca_v2.json \
    --limit 30 \
    --output ./baseline_comparison_results.json
```

#### 4.3 `scripts/generalization_test.py`
- **기능**: 재료명 없는 명령어로 암묵적 추론 능력 평가
- **테스트 케이스**:
  * 암묵적 추론: "투명한 컵" → glass
  * 모호한 명령어: "조심스럽게" → 낮은 grip force
- **총 10개 테스트 케이스**:
  * 7개 암묵적 재료 추론
  * 3개 맥락 이해

**사용 예시**:
```bash
python scripts/generalization_test.py \
    --adapter_dir ./droid-physics-qwen14b-qlora \
    --test_type all \
    --output ./generalization_test_results.json
```
