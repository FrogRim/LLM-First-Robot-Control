## 연구 개요

### 연구 배경 및 목표
로봇 제어 분야에서 자연어 명령을 통한 직관적인 인터페이스는 중요한 연구 과제입니다. 본 연구는 LLM의 물리 추론 능력을 활용하여 자연어 명령으로부터 물리 속성을 직접 추론하고 로봇 제어 파라미터를 자동 생성하는 시스템을 개발하였습니다.

**주요 목표**:
1. DROID 데이터셋을 Genesis AI 시뮬레이션 환경으로 변환
2. Qwen2.5-14B 모델을 물리 도메인에 맞게 QLoRA 방식으로 학습
3. JSON 파싱률 100% 달성 및 준실시간 제어 파라미터 생성

**연구 범위**: 시뮬레이션 환경에서의 Proof-of-Concept (실제 로봇 검증은 향후 연구)

### 주요 기여점
- **학술적**: 공개 데이터셋 활용 방법론, Baseline 비교를 통한 Fine-tuning 효과 입증 (+14.3%p)
- **기술적**: End-to-End 통합 시스템, 견고한 JSON 파싱 (100%)
- **실용적**: 물류·제조·의료 분야 응용 가능성

## 시스템 설계 및 방법론

### LLM-First 아키텍처
```
자연어 명령 → LLM 추론 → 물리 속성 JSON → 제어 파라미터 → Genesis AI 시뮬레이션
```

**설계 원칙**:
- LLM이 물리 속성을 직접 추론 (중간 매핑 레이어 없음)
- JSON 형식으로 구조화된 출력 보장
- 안전성 우선 (fragile 재료에 대한 자동 safety margin 증가)

### DROID → Genesis AI 변환 파이프라인
- DROID 데이터셋 (76,000 에피소드) → Genesis AI 환경 변환
- 좌표계 변환, 키네마틱 매핑, 물리 속성 매핑 완료
- 변환 성공률: 100% (10개 에피소드 검증)

## 실험 결과

### Baseline 비교 실험
| 지표 | Base Model | Fine-tuned | 차이 |
|------|------------|-----------|------|
| JSON 파싱률 | 100% | 100% | 0% |
| 평균 추론 시간 | 9.3초 | 16.6초 | +78% |
| 평균 신뢰도 | 0.850 | 0.853 | +0.003 |

**발견**: 명시적 재료 명령어에서는 두 모델 모두 100% 성능

### 일반화 테스트 (암묵적 재료 추론)
| 지표 | Base Model | Fine-tuned | 개선 |
|------|------------|-----------|------|
| 재료 인식 정확도 | 57.1% | 71.4% | **+14.3%p** |
| 재료 구체성 | 일반적 | 하위분류 포함 | 질적 향상 |

**핵심 발견**:
- Fine-tuning으로 도메인 지식 습득 확인
- 실세계 시나리오(재료명 미명시)에서 명확한 차이
- 구체적 재료 추론: "tempered_glass", "steel" 등

### 주요 성능 지표
- **JSON 파싱률**: 100% (95% CI: 88.4%-100%)
- **추론 시간**: 15.6초 (RTX 4060 Ti 하드웨어 제약, 최적화 후 39% 개선)
- **재료별 정확도**: 100% (7가지 재료 모두)
- **시뮬레이션 검증**: 100% 성공 (4개 시나리오)
- **제어 파라미터 생성**: 0.4ms (목표 대비 500배 빠름)

## 결론

본 연구는 LLM-First 아키텍처 기반의 물리 인식 로봇 제어 시스템을 성공적으로 구축하였습니다. **Baseline 비교 실험을 통해 Fine-tuning의 효과를 정량적으로 입증**하였으며, 특히 재료명이 명시되지 않은 실세계 시나리오에서 +14.3%p의 유의미한 성능 향상을 확인하였습니다.

**주요 성과**:
1. DROID 데이터셋 변환 파이프라인 완성 (100% 성공)
2. 데이터 증강으로 525개 고품질 학습 데이터 생성
3. Qwen2.5-14B 모델 물리 도메인 특화 학습 완료
4. 일반화 능력 입증: 암묵적 재료 추론 71.4% 정확도
5. 통계적 엄밀성 확보 (Bootstrap CI, Perplexity 분석)

**연구 의의**: 대형 LLM이 도메인 특화 학습을 통해 일반화 능력을 향상시킬 수 있음을 실험적으로 입증하였습니다.
